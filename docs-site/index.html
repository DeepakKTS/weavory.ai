weavory.ai

Responsible-AI memory for agent swarms. Every belief signed. Every decision auditable.

Apache 2.0 Responsible AI MCP-native

Agents in regulated industries can't be trusted without a paper trail. When an agent denies a claim, nothing signed that. When it changes its mind, nothing records the history. When compliance asks what it knew on Tuesday, nothing answers. weavory fixes that in five MCP tools — and works with any MCP-native agent.

Live demo

Scrub a BFSI claims-triage session in your browser — watch signed beliefs arrive, trust cells light up, a quarantine LED flash on forged input, and replay exactly what was known at any past instant.

Three ways to install

Fastest · one command, no clone, no build
npx -y @weavory/mcp start
Container · multi-arch (amd64, arm64)
docker run -v weavory-data:/data ghcr.io/deepakkts/weavory:latest
From source
git clone https://github.com/DeepakKTS/weavory.ai.git
cd weavory.ai && pnpm install && pnpm build
